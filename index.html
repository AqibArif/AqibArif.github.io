Deputed to the Pakistan Stock Exchange as Software Developer (Mar 2022), Aqib played a pivotal role building the Jade Trading Terminal (JTT) — a core Order Management System within PSX's New Trading and Surveillance System.

Delivered components including:

  • Validation Server — resilient TCP server handling thousands of concurrent client requests with stringent user-validation for secure trading access.
  • Market by Symbol (MBS) — FIX protocol connections to the Market Data Gateway for tick-by-tick feeds, with an in-memory mapping system delivering real-time order book depth via TIBCO message queues.
  • ETF Circuit Breaker — price-limit calculation server based on live KIBOR rates.
  • CDC Interface — RESTful APIs for end-of-day share settlement with the Central Depository Company.
  • KITS — upgraded the Karachi Internet Trading System for compatibility with the new platform and PSX risk-management rules.
